4. dict的一些特性及注意事项

	* 键必须是可以散列的
	* 字典在内存上的开销巨大
	* 键查询很快
	* 键的次序取决于添加顺序
	* 往字典里添加新键可能会改变已有键的顺序


5. set的实现以及导致的结果

	* 集合里的元素必须是可散列的
	* 集合很消耗内存
	* 可以很高效的判断元素是否存在于某个集合
	* 元素的次序取决于被添加到集合里的次序
	* 往集合里添加元素，可能会改变集合里已有元素的次序
